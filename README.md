# arithmetic_arranger

Function that receives a list of strings that are arithmetic problems and returns the problems arranged vertically and side-by-side.

# Example: 

Input: 

arithmetic_arranger(["32 - 698", "1 - 3801", "45 + 43", "123 + 49"], True))

Output: 

   32         1      45      123
- 698    - 3801    + 43    +  49
-----    ------    ----    -----
 -666     -3800      88      172
